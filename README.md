# Kali-Linux-Security-Lab

## Project Overview
This project demonstrates the installation and use ofKali Linux in a virtual machine envoirement.It includes basic Linux commands and introuctory network scanning using security tools.

## Skills Demonstrated
- Linux command line usage
- Virtual machine setup (Kali Linux)
- Basic networking concepts
- Network scanning using Nmap
- Troubleshooting and system navigation

## Tools Used
- Kali Linux
- VitualBox  / VMware
- Bash Shell
- Nmap

## Tasks Performed
- Installed Kali Linux on a virtual machine
- Navigated the system using basic Linux commands
- Checked network configuration using `ip a`
- Performed basic scans using Nmap
- Practiced terminal commands and system interaction

 ## What i learned 
 This project helped me understand how to work with Kali Linux , use securitytools like Nmap, and perform basics networkscanning. These skills are important for IT support and entry-level cybersecurity roles.
